Operation

The processor is capable of performing six operations: No-op, load, store, add, subtract, and halt.

No-op (No Operation)
-Does nothing, used to stall
Load
-Load data from Data Memory to Register
Store
-Store data from Register into Data Memory
Add
-Adds two data values from Register and store result in Register
Subtract
-Subtracts two data values from Register and store result in Register
Halt
-Disables the Processor. Processor can only be reaabled if reset
Instructions are read from a 128x16 ROM module.
